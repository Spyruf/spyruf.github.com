Welcome to UpNext!

It's pretty simple to use

◌ Search for music using the search bar
◌ Click a song to add it to your queue
◌ To change the order simply drag and drop tracks in the queue

Listening with friends? Tracks will be in sync!

◌ Changes in the current track or queue will be synced across users
◌ Current number of users will be displayed at the top right

Previous Track (Press twice if a track has already started)

Next Track
